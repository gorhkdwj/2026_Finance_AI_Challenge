# P1 — 공고 예시 역추출

## 한눈에 보기

- **항목 수:** 20개
- **주요 패턴:** (1) 신규 진입자(외국인 근로자, 창업 초기 소상공인, 초기 벤처기업)는 하나같이 "국내 신용·거래 이력 부재"라는 동일한 구조적 장벽에 부딪힌다. (2) 보이스피싱·대포통장·AI 음성복제는 서로 다른 사건이 아니라 하나의 범죄 인프라(명의도용 계좌)를 공유하는 연쇄 문제다. (3) 디지털 전환(점포 축소, 키오스크·앱 전환)이 고령층·장애인에게는 접근성 상실로 나타난다.
- **특이 사항:** 예시 4(벤처기업 성장성 평가)와 예시 1(창업기업 매칭)은 "재무제표·거래이력 중심 평가가 미래가치를 못 담는다"는 동일한 근본 문제를 서로 다른 당사자(소상공인 vs 벤처기업)에게서 반복 노출했다 — 축 간 중복이 아니라 같은 항 내에서도 근본 원인이 겹친다는 신호로 판단해 그대로 유지했다. URL 자동 확인 실패(수동 확인 완료) 0건.
- **다음 단계 유의점:** P1-09(벤처기업 평가), P1-14(장애인 접근성), P1-17(FDS 오탐), P1-19(AI 레드팀 취약점)은 정량 피해 규모를 찾지 못해 `수치없음`으로 남겼다 — 검증 단계에서 정량 근거를 추가로 찾을 가치가 있다. P1-06의 출처(한국시니어신문)는 지역 매체이나 국회 행정안전위원회가 2025년 9월 공개한 경찰청 자료를 인용한 것으로, 원자료(국회 의원실 보도자료)를 직접 찾으면 더 견고해진다.
