# P4 — 채널·프로세스 마찰

## 한눈에 보기

- **항목 수:** 12개 (창구 2 / 모바일앱 2 / 인터넷뱅킹·PC웹 2 / 콜센터·ARS 2 / 오픈뱅킹 2 / 마이데이터 2)
- **주요 패턴:** ① 오픈뱅킹·마이데이터 모두 "가입은 쉽지만 해지는 어렵다"는 동일한 구조적 마찰이 반복 확인된다(오픈뱅킹 해지 어려움 62.5%, 마이데이터 해지 어려움 60.0%, 동일한 한국소비자원 조사에서 도출). ② 대면 채널(창구) 축소와 비대면 채널의 접근성 미비가 동시에 발생해 고령층·장애인이 이중으로 소외된다(가평·포천·연천 영업점 2개, 화면낭독기 오류, 70대 모바일뱅킹 이용률 9%). ③ 콜센터·ARS는 다단계 메뉴 구조와 "직원연결 불가" 안내 반복이라는 공통 마찰점이 있으며, 인터넷은행에서 관련 민원 비중이 특히 높다(4대 은행 3단계, 인터넷은행 3사 민원의 36.1%).
- **특이 사항:** P4-04·P4-10·P4-12 세 항목은 동일 출처(한국소비자원 "2022년 모바일 앱 접근성 실태조사", 경기일보 2023-12-21 보도)에서 서로 다른 사실을 각각 인용했다. 조사 시점이 2022년으로 다소 오래됐으나 이후 갱신된 최신 조사를 찾지 못해 "2022년 기준"임을 각 항목에 명기했다. P4-02·P4-05는 동일한 헤럴드경제 기사(2024-11-18)에서 서로 다른 두 사실(점포·ATM 감소 vs 고령층 이용률·혜택손실)을 각각 인용했다. URL 자동 확인 결과는 아래 검증 절차에서 별도 기재.
- **다음 단계 유의점:** P4-04·P4-08·P4-10·P4-12는 기준 데이터가 2022년 것이므로 후속 단계에서 최신 실태조사로 갱신 여부를 확인할 필요가 있다. P4-09(오픈뱅킹 공동망 장애)는 피해 금액·건수가 확인되지 않아 `수치없음`으로 표기했으므로 피해 규모 보강이 필요하면 별도 조사가 필요하다. P3(소비자 페르소나) 축의 장애인·고령층 항목과 일부 주제가 겹치나 헌장 §6에 따라 중복을 허용했다.

### P4-01

- 당사자: 경기 가평·포천·연천군에 거주하는 고령 금융소비자

- 문제 한줄: 마이데이터 서비스를 해지하려는 은행 앱 이용자가 가입 때와 달리 연동 자산을 개별적으로 선택해야만 해지할 수 있어 정보 연동을 끊지 못한다
- 당사자: 마이데이터 서비스 해지를 시도하는 은행 모바일 앱 이용자
- 발생 맥락: 여러 금융사에 흩어진 자산 정보를 연동해둔 마이데이터 서비스를 해지하려는 시점
- 피해 규모: 조사 대상 5개 은행 중 3개(60.0%)가 가입보다 해지가 더 어려운 구조였고, 5개 중 3개는 개인신용정보 수집 관련 주의사항을 화면 하단에 배치해 소비자가 인식하기 어려움(한국소비자원 2022년 조사, 2023-12-21 보도)
- 현재 대응: 수치없음(소비자원이 개선을 권고하는 수준)
