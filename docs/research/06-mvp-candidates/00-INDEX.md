# MVP 후보 15개 — 팀 브레인스토밍 인덱스

- 작성 목적: 5개 주제 × 3개 MVP = 15개 후보를 팀이 한눈에 비교하고 주제별 1~2개씩 추리기 위함
- 각 MVP 상세(지정 양식 12항목)는 주제별 파일 참조
- 근거: 모든 MVP는 `docs/research/05-shortlist.md`·`03-problem-validation.md`의 검증된 문제·수치에 묶여 있음

## 데이터 확보성 범례
- 🟢 공개 데이터/API 확인됨 (바로 착수 가능)
- 🟡 공개 데이터 존재하나 신청·라이선스 미검증
- 🔴 실데이터 비공개 → 합성/공개 학술 데이터로 데모

## 팀 적합도 참고 (ML·LLM·도메인 강점 / 웹배포 약점)
- UI 부담 낮음 = 팀 약점 회피 유리
